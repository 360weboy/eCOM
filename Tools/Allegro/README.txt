Dla poprawnego działania tego skryptu potrzebny jest klucz Allegro WebAPI. Aby go otrzymać należy postępować zgodnie z instrukcjami zawartymi na stronie http://allegro.pl/webapi/
Klucz należy wpisać do zmienej API_KEY w 9 linijce skryptu.

This script needs Allegro WebAPI key. If you don't have one you can generate it following instructions described on page http://allegro.pl/webapi/
You should enter your WebAPI key into variable API_KEY (line 9)
