Minimum requirements

To run our application, you need a Android 2.1 or higher device. To function properly, internet access is required.

Configuring plugins

osCommerce plugin:

You probably have osCommerce up and running by now, you just need to make some changes:

  1. Unzip file oscommerceplugin.zip
  2. Copy files:
    • generate_order_list.php >> /your_main_oscommerce_catalog/
    • set_plugin_pin.php >> /your_main_oscommerce_catalog/admin/
    • pin.ini >> /your_main_oscommerce_catalog/admin/
  3. Edit files:
    • orders.php >> /your_main_oscommerce_catalog/admin/
      Find section (bottom page): 
      <?php
  require(DIR_WS_INCLUDES . 'template_bottom.php');
  require(DIR_WS_INCLUDES . 'application_bottom.php');
?>

      Insert above require():
      echo '<form action="set_plugin_pin.php" method="post"><input type="submit" value="Set mobile PIN number" />';

      So section would look like:
      <?php
  echo '<form action="set_plugin_pin.php" method="post"><input type="submit" value="Set mobile PIN number" />';
  require(DIR_WS_INCLUDES . 'template_bottom.php');
  require(DIR_WS_INCLUDES . 'application_bottom.php');
?>

    • .htaccess >> /your_main_oscommerce_catalog/admin/
      Add:
      <Files *.ini>
Order Deny,Allow
Deny from all
</Files>

      This will prevent remote access to any .ini files, so no one could see your PIN number.

    You may also copy modified files from archive, but first check if you have the same version of osCommerce. Editing files method works for all versions.
  4. Type your store address and PIN number in apropriate boxes in our mobile app
You can change your PIN number by clicking Set mobile PIN number button in your Admin Panel -> Customers Tab -> Orders.

Magento plugin:

To install our Magento plugin, you need to follow theese steps:

  1. Unzip file magentoplugin.zip
  2. Copy files:
    • generate_order_list.php >> /your_main_magento_catalog/
    • pin.ini >> /your_main_magento_catalog/
  3. Edit file:
    • .htaccess >> /your_main_magento_catalog/
      Add:
      <Files *.ini>
Order Deny,Allow
Deny from all
</Files>

      This will prevent remote access to any .ini files, so no one could see your PIN number.
  4. Type your store address and PIN number in apropriate boxes in our mobile app
You can change your PIN number by editing pin.ini file.

PrestaShop plugin:

To install our PrestaShop plugin you need to do the following:
  1. Unzip file prestashopplugin.zip
  2. Copy:
    • mobileorders >> /prestashop_main_dir/modules/
    • mo.php >> /prestashop_main_dir/
  3. Go to your store's administration panel:
    • choose Modules tab
    • in the Administration section look for Mobile orders entry
    • click Install button on the right
    • After refreshing page click Settings link placed below module title and description
    • put your secret PIN number into visible text field and click Save settings button
  4. Type your store address and PIN number in apropriate boxes in our mobile app

OpenCart plugin:

To install our OpenCart plugin you just need to modify your shop a little:
  1. Unzip file opencartplugin.zip
  2. Copy extracted files into opencart main directory and replace conflicted files.
  3. Run admin/install.php on server.
  4. Enter admin panel->System->User->User Groups.
    Edit 'Top Administrator'
    Select 'tool/mobile_orders' in Access Permissions box and save changes.
  5. Congratulations. You've successfully set up mobile orders system. You can set pin code in admin panel by entering: 'System->Mobile Orders'

Configuring API's

Shopify.com API:

Follow theese steps to configure your Shopify.com shop to use our mobile app:
  1. Type your shop URL into textbox below and click Generate link button
    Shop URL (eg. "superstore.myshopify.com"):
  2. Click or copy into your browser's address bar generated link:
  3. Log in if required
  4. Click Install
  5. Save generated password and type it in Key box in our mobile app providing your shop URL in Shop address box (eg. "superstore.myshopify.com")

Shoper.pl API:

Follow theese steps to configure your Shoper.pl shop to use our mobile app:
  1. Go to your store's administration site
  2. Click on Konfiguracja button
  3. Choose Administracja, system from visible popup menu, then choose Administratorzy
  4. Click on dodaj grupę administratorów link:
    • Fill the form's fields as you wish. From Tryb dostępu list choose option: dostęp do webapi
    • Click on Zapisz button, then go to the Uprawnienia tab (left side)
    • From the Sprzedaż list choose option: odczyt and click the Zapisz i zamknij button on the bottom of page
  5. Click on dodaj administratora link:
    • Fill the form's fields as you wish. From Grupa list choose option which name represents the group you added before.
    • Click the Zapisz i zamknij button on the bottom of page
  6. Type your store address and created administrator login with password in apropriate boxes in our mobile app
