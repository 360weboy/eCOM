Application for mobile phones with Android OS 2.3.3 or newer.

Before using this application you must provide some additional information.
- /assets/www/logger.js
In first line in variable loggerUrl you must store the Url address of logger application.

- /assets/www/orders.js
In first three lines you must write the Url's of Allegro, Shoper and Shopify PHP scripts (can be found in tools directory).
